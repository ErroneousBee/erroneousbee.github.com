This is a wordsort demonstration for editure.co.uk

Latest source available at https://github.com/ErroneousBee/erroneousbee.github.com/tree/master/portfolio/java

To build:
cd to the portfolio/java/ directory
$ ant jar

To see a simple demo
$ ant run

To run the jar with example input
$ java -jar build/jar/WordSort.jar Fish DOG banana fish cat a pineapple carrot dog
